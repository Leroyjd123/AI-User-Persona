# PersonaFlow - AI User Research Tool

PersonaFlow is an AI-powered platform designed to streamline user research by generating detailed user personas and facilitating simulated interviews. Built with React, Vite, and Google's Generative AI, it provides a seamless experience for designers and researchers to validate their product assumptions.

## ‚ú® Features

- **AI Persona Generation**: Instantly create detailed user personas with professional backgrounds, demographics, and psychological profiles.
- **Advanced Customization**: Fine-tune your persona's identity, mindset, and context using sliders and custom traits.
- **Interactive Interview Room**: Chat directly with your generated personas to simulate user interviews and gather qualitative insights.
- **Export to PDF**: Generate professional persona reports ready for sharing with your team.
- **Responsive & Concise Design**: A modern, lightweight interface designed for focus and productivity.

## üöÄ Getting Started

### Prerequisites

- **Node.js** (v18 or higher)
- **NPM** or **Yarn**

### Installation

1.  **Clone the repository**:
    ```bash
    git clone [repository-url]
    cd personaFlow
    ```

2.  **Install dependencies**:
    ```bash
    npm install
    ```

3.  **Set up environment variables**:
    Create a `.env.local` file in the root directory and add your Gemini API key:
    ```env
    VITE_GEMINI_API_KEY=your_api_key_here
    ```

4.  **Run the development server**:
    ```bash
    npm run dev
    ```
